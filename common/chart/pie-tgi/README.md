# pie-tgi

> 数说方舟自定义组件：TGI环形图

## Quick Start

```bash
$ npm i

# 启动本地开发环境
$ npm run dev

# 打包 dist 文件
$ npm run build

# 打包并生成可上传方舟系统的 zip 文件
$ npm run build:lego
```
